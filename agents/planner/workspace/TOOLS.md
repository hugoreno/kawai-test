# Tools & Environment Notes

## Skills
- Workspace skills: `~/.openclaw/skills/calendar-management/SKILL.md`
- Workspace skills: `~/.openclaw/skills/weekly-planning/SKILL.md`
- Workspace skills: `~/.openclaw/skills/morning-briefing/SKILL.md`

## Memory
- Weekly plans: `memory/weeks/YYYY-Www.md` — one file per week (ISO week number)
- Events: `memory/events/YYYY-MM-DD.md` — one file per day with events
- Reminders: `memory/REMINDERS.md` — active reminders and nudges
- Recurring events: `memory/RECURRING.md` — weekly/monthly recurring commitments
- Briefing archive: `memory/briefings/YYYY-MM-DD.md` — past morning briefings

## File Conventions
- Workspace root: `~/.openclaw/workspace-planner`
- Memory root: `~/.openclaw/workspace-planner/memory/`
- All dates in ISO format (YYYY-MM-DD)
- Week identifiers in ISO format (YYYY-Www, e.g., 2026-W11)
- Times in 24h format with timezone (e.g., 14:30 ET)
- Events use structured markdown with time, title, location, notes, and status
