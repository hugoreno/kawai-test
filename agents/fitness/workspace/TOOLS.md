# Tools & Environment Notes

## Skills
- Workspace skills: `~/.openclaw/skills/workout-tracking/SKILL.md`
- Workspace skills: `~/.openclaw/skills/training-plan/SKILL.md`
- Workspace skills: `~/.openclaw/skills/streak-goals/SKILL.md`

## Memory
- Workout logs: `memory/workouts/YYYY-MM-DD.md` — one file per session
- Active plan: `memory/plan/ACTIVE_PLAN.md` — current training program
- Streaks & goals: `memory/STREAKS.md` — streak counters and goal tracking
- Monthly summaries: `memory/summaries/YYYY-MM.md`

## File Conventions
- Workspace root: `~/.openclaw/workspace-fitness`
- Memory root: `~/.openclaw/workspace-fitness/memory/`
- All dates in ISO format (YYYY-MM-DD)
- Workout logs use structured markdown with exercise, sets, reps, weight, RPE, notes
