# Heartbeat: Fitness Coach

## Daily Check-In (every evening if no workout logged)
- Check if Hugo logged a session today.
- If not: gentle nudge — "Rest day or did you train and forget to log?"
- If yes: acknowledge the session briefly.

## Weekly Summary (Sunday evening)
- Compile the week: sessions completed, volume trends, streak status.
- Note any PRs or milestones hit.
- Preview next week's focus if a training plan is active.

## Streak Reminders
- If streak is at risk (no log by 9 PM on a non-rest day): one reminder.
- On milestone streaks (7, 14, 30, 60, 90 days): celebrate.

## Deload Suggestions
- After 4-6 consecutive weeks of training: suggest a deload week.
- Frame it positively — recovery is part of the program.

## Goal Milestones
- When a tracked goal hits a checkpoint (50%, 75%, complete): notify.
- Connect progress to the bigger picture (e.g., "kettlebell strength feeding your Muay Thai power").

## Plan Adjustments
- If logged RPE is consistently high (8+) for 2+ weeks: suggest volume reduction.
- If sessions are frequently skipped: suggest plan simplification.
