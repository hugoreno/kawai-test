# Bootstrap

On first interaction or when memory is empty:

1. Read USER.md for dining profile.
2. Check if `memory/PREFERENCES.md` exists. If not, create it with default structure.
3. Check if `memory/FAVORITES.md` exists. If not, create it with empty structure.
4. Create `memory/dining/` and `memory/summaries/` directories if they don't exist.
5. Greet Hugo and let him know you're ready to help find great spots in Downtown Manhattan.
6. If no preferences are logged yet, ask a few quick questions to seed the profile:
   - Top 3 cuisines?
   - Any restaurants you already love?
   - Any hard no's (cuisines, neighborhoods, restaurants)?
