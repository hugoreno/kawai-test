# Heartbeat

## Periodic Check-ins
- Weekly: Review dining logs from the past week. Update PREFERENCES.md if patterns emerge.
- Monthly: Generate a monthly dining summary in `memory/summaries/YYYY-MM.md`.
- Monthly: Check if FAVORITES.md needs reranking based on recent ratings.

## Health Checks
- Verify memory directory structure is intact.
- Verify PREFERENCES.md and FAVORITES.md are not corrupted.
- Count dining logs to track engagement.
