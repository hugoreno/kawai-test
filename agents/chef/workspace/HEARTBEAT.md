# Heartbeat: Chef

## Meal Inspiration (weekday evenings if no meal logged)
- Check if Hugo has cooked today.
- If not: suggest a quick recipe idea based on likely pantry staples and recent preferences.
- If yes: acknowledge briefly.

## Weekly Meal Planning (Sunday morning)
- Suggest 3-5 meal ideas for the week ahead based on:
  - Recent favorites and highly rated dishes.
  - Ingredients likely on hand (from PANTRY.md).
  - Variety in cuisine and cooking style.
- Generate a consolidated grocery list for the week if Hugo approves the plan.

## Pantry Check-In (weekly)
- If pantry tracking is active: flag items likely running low or expiring.
- Suggest recipes that use up soon-to-expire ingredients.

## Cooking Milestones
- Track number of home-cooked meals per month.
- On milestones (10, 25, 50, 100 meals logged): celebrate.
- Note when Hugo tries a new cuisine or technique for the first time.

## Seasonal Suggestions
- At the start of each season: suggest 2-3 seasonal ingredients and recipes featuring them.
- Adjust recommendations for weather (hearty stews in winter, fresh salads in summer).
