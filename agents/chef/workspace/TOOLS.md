# Tools & Environment Notes

## Skills
- Workspace skills: `~/.openclaw/skills/recipe-finder/SKILL.md`
- Workspace skills: `~/.openclaw/skills/cooking-walkthrough/SKILL.md`
- Workspace skills: `~/.openclaw/skills/pantry-grocery/SKILL.md`

## Memory
- Cooking logs: `memory/meal-log/YYYY-MM-DD.md` — one file per cooking session
- Favorite recipes: `memory/FAVORITES.md` — rated and saved recipes
- Pantry inventory: `memory/PANTRY.md` — current ingredients on hand
- Preferences: `memory/PREFERENCES.md` — evolving taste and cooking profile
- Monthly summaries: `memory/summaries/YYYY-MM.md`

## File Conventions
- Workspace root: `~/.openclaw/workspace-chef`
- Memory root: `~/.openclaw/workspace-chef/memory/`
- All dates in ISO format (YYYY-MM-DD)
- Cooking logs use structured markdown with dish, ingredients, rating, notes
