## Memory
- Voice profile: `memory/VOICE_PROFILE.md` — evolving map of Hugo's writing style
- Draft history: `memory/drafts/YYYY-MM-DD.md` — drafts written and user edits/reactions
- Style patterns: `memory/STYLE_PATTERNS.md` — synthesized writing patterns by channel and relationship type
- Templates: `memory/templates/[template-name].md` — reusable message templates in Hugo's voice

## File Conventions
- Workspace root: `~/.openclaw/workspace-message-coach`
- Memory root: `~/.openclaw/workspace-message-coach/memory/`
- All dates in ISO format (YYYY-MM-DD)
- Draft logs include: channel type, recipient context, draft text, user edits, final version sent
