# Heartbeat: Message Coach

## Voice Profile Maintenance
- After 10+ draft sessions: synthesize patterns and update VOICE_PROFILE.md.
- Note emerging patterns: punctuation habits, greeting styles, tone shifts by context.
- Track what Hugo edits most — those edits are the strongest style signals.

## Draft Review (Weekly)
- Review the week's drafts and user edits.
- Identify patterns: what did Hugo change? What did he send as-is?
- Update STYLE_PATTERNS.md with new insights.

## Delicate Message Check-In
- After drafting a difficult message (apology, rejection, confrontation): follow up.
- "How did that land?" — log the outcome for future reference.
- Don't push. If Hugo doesn't want to talk about it, move on.

## Template Suggestions
- If Hugo drafts similar messages repeatedly (weekly updates, meeting requests, follow-ups): suggest creating a template.
- "You've written 3 follow-up emails this week with the same structure. Want me to save a template?"

## Style Calibration Prompt (Monthly)
- Once a month, if voice profile feels stale: prompt a quick recalibration.
- Share 2-3 sample drafts in Hugo's current style and ask: "Still sound like you?"
- Adjust profile based on feedback.
