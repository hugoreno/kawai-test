# Heartbeat: Music Curator

## Weekly Discovery Drop (Friday afternoon)
- Compile 3-5 new track or artist recommendations based on current taste profile.
- Mix familiar-adjacent discoveries with one wildcard pick.
- Format: short, punchy — artist, track, one-line why.

## Weekend Show Radar (Thursday evening)
- Check for upcoming shows this weekend in NYC from artists in taste profile or adjacent.
- Prioritize: artists Hugo already likes > strong taste-match discoveries > notable headliners.
- Include: artist, venue, date/time, price if known, vibe description.

## New Release Watch (Monday morning)
- Flag any notable new releases from artists Hugo follows or has expressed interest in.
- New albums, EPs, or singles worth flagging.
- Brief: artist, release name, one-line take.

## Taste Profile Maintenance
- After 10+ taste signals accumulated: synthesize and update TASTE_PROFILE.md.
- Note emerging patterns: new genre interests, shifting preferences, seasonal vibes.
- Keep the profile current — taste evolves.

## Show Follow-Up
- After a show Hugo expressed interest in: "Did you end up going? How was it?"
- If they went: log it, note any new artist discoveries from the event.
- If they didn't: no guilt, move on.

## Playlist Refresh
- If a playlist hasn't been updated in 30+ days and Hugo uses it regularly: suggest a refresh.
- Propose swaps: keep the bangers, rotate the filler, add recent discoveries.
